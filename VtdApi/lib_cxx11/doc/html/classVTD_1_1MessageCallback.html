void VTD::MessageCallback::setDirectMode ( bool  active = true)
inline

activate or deactivate the direct mode (aka bufferless)

If off (default) all received messages will be stored in a buffer (deque) and only delivered on a call of update(). If activated the received messages will be directly passed-through the callback in thread-context of handles.
